# Zenith Photo - Cinematic Cityscape Archive

![Zenith Photo Preview](https://images.unsplash.com/photo-1449824913935-59a10b8d2000?auto=format&fit=crop&q=80&w=1200)

## 概要 | Overview
**Zenith Photo** は、没入感（イマージョン）を極限まで追求したシネマティックな街歩き映像を制作するクリエイターのポートフォリオサイトです。
撮影者の存在を感じさせず、視聴者がその場所に立っているかのような体験を届けるアーカイブを管理・展示します。

This is a cinematic portfolio website for "Zenith Photo," a creator dedicated to 4K immersive city-walk experiences. The site showcases high-quality archives with a minimalist aesthetic, focusing on pure visual beauty.

## 主な機能 | Features
- **シネマティックなUI/UX**: `Bodoni Moda` フォントとダークモードを採用し、映像の美しさを際立たせるデザイン。
- **スクロールアニメーション**: 独自の `ScrollReveal` コンポーネントによる滑らかなコンテンツ表示。
- **レスポンシブデザイン**: スマートフォンからデスクトップまで、あらゆるデバイスで没入感を損なわない設計。
- **管理ダッシュボード**: 管理者専用ページから、ギャラリー画像やヒーロー画像の動的な管理が可能（Local Storageを使用）。
- **ギャラリーアーカイブ**: 日本各地の風景を切り取った高画質なフォトアーカイブ。

## 技術スタック | Tech Stack
- **Frontend**: React 19, TypeScript
- **Styling**: Tailwind CSS
- **Routing**: React Router (HashRouter)
- **Icons**: Lucide React
- **Animations**: CSS Transitions & Intersection Observer API

## セットアップ | Setup
ローカル環境で実行する場合は、以下の手順に従ってください。

```bash
# リポジトリをクローン
git clone https://github.com/あなたのユーザー名/zenith-photo.git

# ディレクトリに移動
cd zenith-photo

# 依存関係をインストール
npm install

# 開発サーバーを起動
npm run dev
```
